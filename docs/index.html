21 MCP Tools

üì¶ Item Operations

  • sitecore_get_item
    Retrieve a single content item by path or GUID. Returns complete item metadata including template info, display name, language, version, and hasChildren flag. Supports multi-language queries.
  • sitecore_get_children
    Fetch all direct child items of a parent item. Includes pagination support (first/after parameters) and optional recursive traversal for deep tree navigation.
  • sitecore_get_field_value
    Extract the value of a specific field from an item. Lightweight query for single field retrieval without fetching entire item data.
  • sitecore_get_item_fields
    Get all fields (both own and inherited) from an item's template. Returns field names and values in a structured format. Template-aware field discovery with Helix base template support.
  • sitecore_query
    Execute custom Sitecore fast queries using XPath-like syntax (e.g., /sitecore/content//*[@@templatename='Page']). Supports complex filtering and template-based searches.
  • sitecore_search
    Powerful search with filters (path_contains, name_contains, template_in, hasChildren, hasLayout), ordering by name/path/displayName, and faceted search capabilities.
  • sitecore_search_paginated
    Search with cursor-based pagination. Returns pageInfo with hasNextPage/hasPreviousPage flags and totalCount. Use endCursor for next page fetching.

üìã Template Operations

  • sitecore_get_template
    Retrieve complete template definition including all sections, fields, field types, and base template inheritance chain. Essential for understanding item structure in Helix architecture (Foundation/Feature/Project layers).
  • sitecore_get_templates
    Fetch multiple template definitions with optional path filtering (e.g., /sitecore/templates/User Defined). Returns template hierarchies with field definitions for batch template analysis.

üïê Version Control

  • sitecore_get_item_versions
    Retrieve complete version history for an item in a specific language. Returns all version numbers with metadata, enabling version comparison and audit trail analysis.
  • sitecore_get_item_with_statistics
    Get item data enriched with Statistics inline fragment: Created/Updated timestamps, CreatedBy/UpdatedBy user names, and Revision ID. Perfect for audit logs and change tracking.

üß≠ Navigation

  • sitecore_get_parent
    Navigate one level up in the content tree. Returns the immediate parent item with full metadata. Useful for breadcrumb generation and upward tree traversal.
  • sitecore_get_ancestors
    Get complete ancestor path from current item to /sitecore root. Returns ordered array of all parent items, grandparents, etc. Essential for full breadcrumb trails and hierarchical context.

üé® Layout & Sites

  • sitecore_get_layout
    Retrieve complete presentation details for a route including all placeholders, renderings, datasources, and rendering parameters. Uses site name and route path for JSS/Headless scenarios.
  • sitecore_get_sites
    Get Sitecore site definitions with filtering by name or current site. Returns site root paths, hostnames, language settings, and system/user site classifications.

‚úèÔ∏è Mutations

  • sitecore_create_item
    Create new content items via GraphQL mutations. Specify name, template path/ID, parent path, language, and optional field values. Returns created item with GUID and full metadata.
  • sitecore_update_item
    Modify existing item properties including name, field values, and version-specific changes. Supports multi-language updates with language and version parameters.
  • sitecore_delete_item
    Remove items from Sitecore with deletePermanently flag. False moves to recycle bin (soft delete), true removes permanently. Requires appropriate permissions.

üî¨ Advanced Features

sitecore_scan_schema
Perform GraphQL introspection to discover all available queries, mutations, types, and fields. Analyzes schema capabilities and saves comprehensive type definitions to JSON for reference.
sitecore_command
Natural language interface that interprets plain English commands and executes appropriate MCP tools. Examples: "get item /sitecore/content/Home", "search for articles", "find items with template Page".
sitecore_discover_item_dependencies
Comprehensive discovery tool (NEW v1.6.0) that retrieves content item with complete dependency graph: template definition, inheritance chain, all fields (own + inherited), associated renderings, and GraphQL resolvers. Opt-in rendering/resolver discovery for performance.

Quick Start

1. Install Dependencies

npm install
npm run build

2. Configure Environment

# .env
SITECORE_HOST=https://your-sitecore-instance.com
SITECORE_ENDPOINT=https://your-sitecore-instance.com/sitecore/api/graph/items/master
SITECORE_API_KEY=your-api-key-here
SITECORE_USERNAME=sitecore\admin
SITECORE_PASSWORD=your-password

3. Run Tests

.\scripts\tests\run-tests.ps1
# Expected: 17/17 tests passed (100%)
